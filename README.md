# Gravity Siege

Este é um jogo de artilharia por turnos em 2D, construído com HTML5, CSS3, JavaScript e o motor de física Matter.js.

## Como Jogar
1. Abra o arquivo `index.html` em seu navegador ou publique no GitHub Pages.
2. Selecione a quantidade de jogadores e personagens.
3. No seu turno, clique perto de um dos seus personagens e arraste para o lado oposto à direção que deseja atirar (como um estilingue).
4. Solte para disparar o projétil.
5. Destrua o cenário e acerte os oponentes para reduzir o HP deles.
6. O último jogador com personagens no mapa vence!

## Estrutura do Projeto
- `index.html`: Estrutura principal e UI.
- `style.css`: Estilização premium e efeitos visuais.
- `game.js`: Lógica do jogo, física, turnos e destruição.

## Tecnologias Utilizadas
- **Matter.js**: Motor de física para gravidade e colisões.
- **HTML5 Canvas**: Renderização do cenário e projéteis.
- **CSS Glassmorphism**: Interface moderna e translúcida.

## Deploy no GitHub Pages
1. Faça o commit de todos os arquivos para o seu repositório.
2. No GitHub, vá em **Settings** > **Pages**.
3. Selecione a branch `main` e a pasta `/ (root)`.
4. Salve e aguarde o link de publicação.
